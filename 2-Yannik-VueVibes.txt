Vue Vibes
=========

Tired of making too many querySelectors
and managing events of HTML elements?
Check out VUE, the JS framework for perfection
Developing pages is intense!

All elements that act together
are grouped in single file components
Thus all you code is seperated, however
all your components can talk with events.

Define props to parse on information
from an upper parent to a child component
manage your states with data variables
and don't forget to mind your moment!
